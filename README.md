# Atlanta WaterReady — GitHub Pages Ready

This package is ready to publish on GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `script.js`
- `assets/`
- `.nojekyll`

## Easiest deploy options

### Option A: project site
1. Create a new GitHub repository.
2. Upload all files from this folder to the repo root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/ (root)`.
6. Save.

Your site will publish at a URL like:
`https://YOUR-USERNAME.github.io/REPO-NAME/`

### Option B: root github.io site
1. Create a repository named exactly:
`YOUR-USERNAME.github.io`
2. Upload all files from this folder to that repo.
3. Go to **Settings → Pages** and deploy from the `main` branch root.

Your site will publish at:
`https://YOUR-USERNAME.github.io/`

## Notes
- This site uses only static HTML, CSS, JavaScript, and local images.
- All asset paths are relative, so it works for both a root site and a project site.
- `.nojekyll` is included to avoid GitHub Pages processing surprises.
